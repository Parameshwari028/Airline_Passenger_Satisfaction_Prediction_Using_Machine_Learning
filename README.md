    ✈️ Airline Passenger Satisfaction Prediction
This project focuses on predicting airline passenger satisfaction using Machine Learning techniques. The model classifies whether a passenger is Satisfied or Dissatisfied based on various service and travel-related features.
________________________________________
     Project Overview
Airline companies aim to improve customer experience. This project builds a predictive model that helps        identify key factors influencing passenger satisfaction.
________________________________________
     Objectives
1.	Analyze airline passenger data
2.	Perform data preprocessing and feature engineering
3. Train a Machine Learning model (Random Forest)
4.	Evaluate model performance
5.	Predict passenger satisfaction
________________________________________
     Technologies Used
1.	Python
2.	Pandas
3.	NumPy
4.	Scikit-learn
5.  Matplotlib / Seaborn
6.	Google colab
________________________________________
     Dataset Features
Some important features used:
1.	Age
2.	Flight Distance
3.	Class (Business/Economy)
4.	Seat Comfort
5.	Food & Drink
6.	Online Boarding
7.	Check-in Service
8.	Cleanliness
